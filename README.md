### GOF Design Patterns in Java

> The Gang of Four (GOF) design patterns are a set of 23 classic software design patterns introduced by Erich Gamma, Richard Helm, Ralph Johnson, and John Vlissides in their book "Design Patterns: Elements of Reusable Object-Oriented Software." These patterns are categorized into three main groups:

### 1. [Creational Patterns](src/main/java/creational)
### 2. [Structural Patterns](src/main/java/structural)
### 3. [Behavioral Patterns](src/main/java/behavioral)

## Here's a list of all 23 GOF design patterns:

### Creational Patterns:

1. [Singleton](src/main/java/creational/singleton)
2. [Factory Method](src/main/java/creational/factory/method)
3. [Abstract Factory](src/main/java/creational/abstractFactory)
4. [Builder](src/main/java/creational/builder)
5. [Prototype](src/main/java/creational/prototype)

### Structural Patterns:

6. [Adapter](src/main/java/structural/adapter)
7. [Bridge](src/main/java/structural/bridge)
8. [Composite](src/main/java/structural/composite)
9. [Decorator](src/main/java/structural/decorator)
10. [Facade](src/main/java/structural/facade)
11. [Flyweight](src/main/java/structural/flyweight)
12. [Proxy](src/main/java/structural/proxy)

### Behavioral Patterns:

13. [Chain of Responsibility](src/main/java/behavioral/chain/of/responsibility)
14. [Command](src/main/java/behavioral/command)
15. [Interpreter](src/main/java/behavioral/interpreter)
16. [Iterator](src/main/java/behavioral/iterator)
17. [Mediator](src/main/java/behavioral/mediator)
18. [Memento](src/main/java/behavioral/memento)
19. [Observer](src/main/java/behavioral/observer)
20. [State](src/main/java/behavioral/state)
21. [Strategy](src/main/java/behavioral/strategy)
22. [Template Method](src/main/java/behavioral/template/method)
23. [Visitor](src/main/java/behavioral/visitor)

> Each of these patterns addresses a specific design problem and provides a general reusable solution for it. They help in creating more flexible, reusable, and maintainable software designs by promoting loose coupling, encapsulation, and the separation of concerns.
